## 代码运行

1. 创建虚拟环境

```
python -m venv venv
```

2. 激活虚拟环境

```
source venv/bin/activate
```

3. 安装依赖

```
pip install -r requirements.txt
```

5. 保存依赖

```
pip freeze > requirements.txt
```
